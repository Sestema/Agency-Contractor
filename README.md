# Agency Contractor
🇺🇦 Українська | [🇨🇿 Česky](README.cs.md)
**Система управління працівниками, документами, зарплатами та обліком для кадрових агентств**
**Остання стабільна версія:** `0.1.72`  
**PostgreSQL multi-PC:** доступно у стабільному каналі  
**Скачати:** [GitHub Releases](https://github.com/Sestema/Agency-Contractor/releases/latest)
Agency Contractor — це професійне десктопне WPF-застосування для кадрових (трудових) агентств, яке автоматизує повний цикл роботи: від обліку працівників, документів і фірм до генерації договорів, зарплат, звітів, фактур, AI-помічника, Telegram-бота та роботи кількох ПК з однією базою.
---
## Основні можливості
### Облік працівників / Evidence zaměstnanců
- Покроковий wizard додавання працівника (фото, паспорт/ID, віза/карта побиту, страхівка, дозвіл на роботу)
- Детальна картка працівника з вкладками: документи, профіль, історія, зарплата
- Особисті дані, контакти, адреса, громадянство, місце народження
- Дані по документах: номер візи, дата початку/кінця візи, орган видачі візи
- Прив’язка до фірми, агенції, позиції, дати наступу до роботи та завершення
- Активні, архівні, відновлені та нещодавно видалені працівники
- Пошук, фільтри, групування по фірмах, 4 режими перегляду
- Автоматичне кадрування та обробка фото документів
- Історія змін у профілі працівника
### Документи і контроль термінів
- Контроль термінів дії паспорта, візи/карти побиту, страхівки, дозволу на роботу
- Підтримка власних/кастомних документів
- Попередження Warning / Critical / Expired
- Snooze проблем документів на 7/14/30/60/90 днів
- Заміна документів із автоматичною архівацією старих файлів
- Швидке відкриття папки працівника з усіма файлами
- PDF-звіти по проблемних документах
### Управління фірмами, агенціями та кандидатами
- Додавання та налаштування фірм-замовників
- IČO, юридична адреса, робочі адреси, позиції, ставки, графік роботи
- Прив’язка агенції до кожної фірми
- Drag & drop сортування фірм у бічному меню
- Можливість приховати неактивні фірми без видалення
- Кольорове маркування фірм
- Окрема база кандидатів з документами, фото та пошуком
### Шаблони документів
- Редактор DOCX-шаблонів з підтримкою тегів `${TAG_NAME}`
- Редактор XLSX-шаблонів
- Редактор PDF з розміщенням тегів
- Каталог доступних тегів з пошуком
- Стартові шаблони для швидкого початку
- Копіювання шаблонів між фірмами
- **AI автоматичне вставлення тегів** (Google Gemini) — AI аналізує документ і вставляє потрібні змінні
### Генерація документів
- Автоматична генерація DOCX, PDF, XLSX з шаблонів
- Заповнення тегів даними працівника, фірми, агенції, документів і зарплати
- Підтримка чеських трудових договорів з правильним маппінгом полів
- Масова генерація документів для багатьох працівників
- Генерація документів в одну вибрану папку
- Детальний результат генерації по кожному працівнику
### Фінанси та виплати
- Помісячний облік зарплат, годин, ставок, gross/net, авансів і доплат
- Статус виплати: виплачено / не виплачено
- Кастомні колонки у зарплатній таблиці
- Витрати фірм
- Примітки з автоматичним перенесенням на наступні місяці
- Перенесення боргів/залишків між місяцями
- Історія зарплат у картці працівника
- Захист від дублювання історії виплат
- Робота в одному місяці по різних фірмах з меншим ризиком конфліктів
- Підписні листи, PDF та Excel експорт зарплат
### Звіти та аналітика
- Звіти по працівниках, фірмах, агенціях і архіву
- Детальний список працівників і зведені підсумки
- Вибір колонок у звітах
- Колонки: номер візи, орган видачі візи, громадянство, дати документів, фірма, агенція, статус
- Фільтри по датах, фірмах та агенціях
- Виділення архівованих працівників кольором
- Графіки та статистика
- Експорт у PDF і XLSX
### Dashboard
- Головна панель зі статистикою
- Огляд по фірмах, працівниках, документах і зарплатах
- Попередження про терміни дії документів
- Рух працівників за місяць: нові, завершені, архівовані, відновлені
- Зарплатні підсумки по місяцях
- Статистика по фірмах
- AI-звіт по стану агентства
### Рахунки-фактури
- Фактури, цінові пропозиції, замовлення, касові ордери
- ARES інтеграція по IČO
- QR-платежі
- PDF-шаблони рахунків
- Валюти CZK, EUR, USD, PLN
- Каталог клієнтів, постачальників і позицій
### AI-інтеграція (Google Gemini)
- AI чат для консультацій
- AI-сканування документів
- AI-пошук працівників природною мовою
- Автоматичне вставлення тегів у шаблони документів
- Розпізнавання полів чеських трудових договорів
- AI-звіт Dashboard
- AI-переклад і аналіз HR-новин
### Telegram-бот
- Пошук працівників з телефону
- Запити по документах, зарплатах, фірмах і звітах
- Голосові повідомлення і текстові питання
- Контекст розмови: бот пам’ятає останнього знайденого працівника
- Зарплата за місяць, статус виплати, аванси
- Документи, що закінчуються
- Аналітика по працівниках і фірмах
### PostgreSQL multi-PC
- SQLite режим для одного ПК
- PostgreSQL режим для 3–5+ користувачів
- Міграція SQLite → PostgreSQL
- Підключення інших ПК до спільної бази
- Копіювання даних підключення для другого ПК
- Список підключених ПК/користувачів
- Live sync через PostgreSQL LISTEN/NOTIFY
- Автоматичне перепідключення після збоїв мережі, VPN або сну ПК
- Робота через локальну мережу або Tailscale
- PostgreSQL може працювати на головному ПК або сервері
### Спільна папка документів
- База даних зберігається в PostgreSQL
- Фото, скани, шаблони і папки працівників залишаються у спільній кореневій папці
- Папку працівника можна відкривати як звичайну папку Windows
- Підходить для SMB/NAS/серверної папки або спільного диска
### Backup
- Автоматичний щоденний backup у `Backup\YYYY-MM-DD`
- Один backup на день для всіх ПК
- У PostgreSQL режимі спочатку оновлюється SQLite-копія з PostgreSQL, потім створюється backup
- Зберігаються останні 7 днів
- Lock захищає від одночасного backup з кількох ПК
- SQLite backup дозволяє швидко повернутися до старого режиму або перенести папку
### Журнал активності
- Логування дій у системі
- Історія змін працівників
- Історія архівації, відновлення, документів і профілю
- Локалізоване відображення записів журналу
- Експорт журналу в Excel

## Налаштування
- **4 мови інтерфейсу:** Українська, Čeština, English, Русский
- **4 теми оформлення:** Light, Dark, Dark Word, Custom
- **Розмір інтерфейсу:** Small, Medium, Large, Extra Large
- **Розмір тексту:** Small, Medium, Large, Extra Large
- **Мова документів:** налаштування окремо від інтерфейсу
- **Видимість фірм:** приховання неактивних фірм
- **Видимість тегів:** налаштування відображення тегів у каталозі
- **Налаштування серверів:** PostgreSQL, Telegram, Web panel
- **Налаштування AI:** ключ Gemini, модель, AI-функції

## Безпека та ліцензування
- **Система ліцензування** з прив'язкою до комп'ютера (Machine ID)
- **HMAC-SHA256 підпис** ліцензійного файлу
- **DPAPI для секретних даних і паролів**
- **PostgreSQL доступ через логін/пароль**
- **Anti-rollback захист** (виявлення зміни системної дати)
- **Підтвердження для небезпечних дій**
- Документи залишаються у вашій папці під вашим контролем
- Підтримка термінів: 30, 90, 180, 362 дні або безліміт
- Активація через файл `activator.key`

## Автооновлення
Програма підтримує автоматичне оновлення через GitHub Releases і Velopack:
1. Відкрийте **Налаштування** → **Оновлення програми**
2. Натисніть **"Перевірити оновлення"**
3. Якщо є нова версія — натисніть **"Встановити"**
4. Програма завантажить оновлення та перезапуститься

## Системні вимоги
- **ОС:** Windows 10 / 11 (x64)
- **Runtime:** .NET 8.0 Desktop Runtime (включено в Setup.exe)
- **Місце на диску:** ~200 МБ
- **Оперативна пам'ять:** 4 ГБ (рекомендовано 8 ГБ)
- **Для multi-PC:** PostgreSQL і спільна коренева папка документів
- **Для віддаленої роботи:** локальна мережа, VPN або Tailscale

## Встановлення
### Варіант 1: Інсталятор (рекомендовано)
1. Завантажте `AgencyContractor-win-Setup.exe` з [Releases](https://github.com/Sestema/Agency-Contractor/releases)
2. Запустіть та дочекайтесь встановлення
3. Програма встановиться в `%LocalAppData%\AgencyContractor`
### Варіант 2: Портативна версія
1. Завантажте `AgencyContractor-win-Portable.zip` з [Releases](https://github.com/Sestema/Agency-Contractor/releases)
2. Розпакуйте в будь-яку папку
3. Запустіть `Win11DesktopApp.exe`
