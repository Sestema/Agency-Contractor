# Agency Contractor 0.1.67 — детальніша панель керування

## Панель керування
- **Рух за місяць відкривається детально** — можна подивитися, хто наступив у поточному місяці, хто закінчив роботу, у якій фірмі та з якою датою.
- **Фірми відкривають детальний список** — показується кількість працівників, проблем з документами та шаблонів по кожній фірмі.
- **Активні працівники відкривають загальний список** — картка активних працівників більше не веде в шаблони, а відкриває список усіх працівників.
- **Клік по рядках відкриває потрібний розділ** — працівник відкриває картку, фірма відкриває список працівників цієї фірми.

## Стабільність
- **Виправлено падіння dashboard** — панель фірм більше не падає через binding до `EmployeeRatio`.
- **Покращено тексти інтерфейсу** — додано локалізовані підписи для нових деталей панелі керування.

## Автооновлення
- **Підготовлено для каналу оновлень** — реліз `0.1.67` готовий до публікації через GitHub Releases і Velopack.
