# Agency Contractor

🇺🇦 Українська | [🇨🇿 Česky](README.cs.md)

**Система управління працівниками та документами для кадрових агентств**

Agency Contractor — це професійне десктопне WPF-застосування для кадрових (трудових) агентств, яке автоматизує повний цикл роботи: від реєстрації працівника до генерації документів, контролю термінів та фінансового обліку.

---

## Основні можливості

### Управління працівниками
- Покроковий wizard додавання працівника (фото, паспорт, віза, страхівка, дозвіл на роботу)
- Автоматичне кадрування та обробка фото документів
- Детальний профіль працівника з історією змін
- Архівування працівників зі збереженням всіх даних
- Контроль термінів дії документів (паспорт, віза, страхівка, дозвіл на роботу)
- Автоматичне архівування старих документів при оновленні

### Управління фірмами
- Додавання та налаштування фірм-замовників
- Drag & drop сортування фірм у бічному меню
- Можливість приховати неактивні фірми
- Кольорове маркування фірм

### Шаблони документів
- Редактор DOCX-шаблонів з підтримкою тегів `${TAG_NAME}`
- Редактор XLSX-шаблонів
- Редактор PDF
- Каталог доступних тегів з пошуком
- **AI автоматичне вставлення тегів** (Google Gemini) — AI аналізує текст документа і автоматично вставляє потрібні теги в правильні місця

### Генерація документів
- Автоматична генерація DOCX, PDF, XLSX з шаблонів
- Заповнення тегів даними працівника та фірми
- Підтримка чеських трудових договорів з правильним маппінгом полів

### Фінанси та виплати
- Облік зарплат, авансів, доплат по місяцях
- Примітки з автоматичним перенесенням на наступні місяці
- Збереження ширини колонок
- Експорт фінансових звітів

### Звіти
- Звіт по працівниках (PDF, XLSX)
- Звіт по фірмах (PDF, XLSX)
- Виділення архівованих працівників кольором
- Вирівнювання та форматування колонок

### Кандидати
- База потенційних працівників
- Окремий розділ для управління кандидатами

### Dashboard
- Головна панель зі статистикою
- Огляд по фірмах та працівниках
- Попередження про терміни дії документів

### AI-інтеграція (Google Gemini)
- AI чат для консультацій
- Автоматичне вставлення тегів у шаблони документів
- Розпізнавання полів чеських трудових договорів

### Журнал активності
- Логування всіх дій в системі
- Історія змін

## Налаштування

- **4 мови інтерфейсу:** Українська, Čeština, English, Русский
- **4 теми оформлення:** Light, Dark, Dark Word, Custom
- **Розмір інтерфейсу:** Small, Medium, Large, Extra Large
- **Розмір тексту:** Small, Medium, Large, Extra Large
- **Мова документів:** налаштування окремо від інтерфейсу
- **Видимість фірм:** приховання неактивних фірм
- **Видимість тегів:** налаштування відображення тегів у каталозі

---

## Безпека та ліцензування

- **Система ліцензування** з прив'язкою до комп'ютера (Machine ID)
- **HMAC-SHA256 підпис** ліцензійного файлу
- **DPAPI шифрування** ліцензійних даних
- **Anti-rollback захист** (виявлення зміни системної дати)
- Підтримка термінів: 30, 90, 180, 362 дні або безліміт
- Активація через файл `activator.key`

## Автооновлення

Програма підтримує автоматичне оновлення через GitHub Releases:
1. Відкрийте **Налаштування** → **Оновлення програми**
2. Натисніть **"Перевірити оновлення"**
3. Якщо є нова версія — натисніть **"Встановити"**
4. Програма завантажить оновлення та перезапуститься

## Системні вимоги

- **ОС:** Windows 10 / 11 (x64)
- **Runtime:** .NET 8.0 Desktop Runtime (включено в Setup.exe)
- **Місце на диску:** ~200 МБ
- **Оперативна пам'ять:** 4 ГБ (рекомендовано 8 ГБ)

## Встановлення

### Варіант 1: Інсталятор (рекомендовано)
1. Завантажте `AgencyContractor-win-Setup.exe` з [Releases](https://github.com/Sestema/Agency-Contractor/releases)
2. Запустіть та дочекайтесь встановлення
3. Програма встановиться в `%LocalAppData%\AgencyContractor`

### Варіант 2: Портативна версія
1. Завантажте `AgencyContractor-win-Portable.zip` з [Releases](https://github.com/Sestema/Agency-Contractor/releases)
2. Розпакуйте в будь-яку папку
3. Запустіть `Win11DesktopApp.exe`

## Технології

- **WPF** (.NET 8.0) — інтерфейс
- **MVVM** — архітектура
- **OpenXML SDK** — робота з DOCX/XLSX
- **PDFsharp** — генерація PDF
- **ClosedXML** — робота з Excel
- **OpenCvSharp** — обробка зображень
- **Google Gemini API** — AI-функції
- **Velopack** — автооновлення
- **DPAPI** — шифрування ліцензій
