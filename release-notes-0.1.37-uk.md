# Agency Contractor 0.1.37

Стабілізаційне оновлення з покращенням доступу, ліцензування та Admin Panel.

- доступ до програми став більш передбачуваним: серверний стан тепер має вищий пріоритет, додано офлайн-пільгу та кращу синхронізацію статусу;
- старі локальні ліцензії тепер можуть одноразово переноситися на сервер, щоб подальше керування доступом працювало централізовано через Admin Panel;
- Admin Panel отримала легший старт, завантаження telemetry тільки для вибраного клієнта, server-side risk summary та пагінацію подій;
- виправлено race-condition `Collection was modified`, видалення фірми з очищенням папки та покращено загальну стабільність старту і синхронізації.
