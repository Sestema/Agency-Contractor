# Agency Contractor 0.1.40

Стабілізаційне оновлення зі швидшим і більш передбачуваним запуском програми.

- прискорено старт застосунку за рахунок спрощення початкової telemetry-перевірки;
- додано безпечний timeout для profile gate, щоб повільна мережа або затримки сервера не блокували відкриття програми;
- покращено загальну стабільність startup flow без змін у логіці доступу, ліцензії та автооновлення.
