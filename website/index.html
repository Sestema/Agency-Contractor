FAQ

Поширені запитання

Чи безпечні мої дані?

+

Абсолютно. Всі дані зберігаються локально на вашому комп'ютері та зашифровані за допомогою алгоритму AES-256 та Windows DPAPI. Ми не маємо доступу до вашої бази даних.

Чи потрібен інтернет для роботи?

+

Основний функціонал програми працює повністю офлайн. Інтернет потрібен лише для функцій AI (Google Gemini), інтеграції з ARES (реєстр Чехії) та перевірки ліцензії.

Як працює 14-денний тестовий період?

+

Ви завантажуєте програму і отримуєте повний доступ до всіх функцій (рівень "Безліміт") на 14 днів. Жодної прив'язки картки. Після завершення періоду ви зможете обрати зручний тариф.

Чому Windows або антивірус блокує файл?

+

Оскільки програма нова і розробник ще не має накопиченого "рейтингу довіри" від Microsoft (SmartScreen), система може попередити про невідомий файл. Це абсолютно безпечно. При завантаженні дозвольте збереження файлу, а при запуску натисніть "Докладніше" (More info) та "Виконати в будь-якому випадку" (Run anyway).

Який саме файл потрібно скачувати?

+

На сторінці GitHub Releases завжди обирайте найновішу версію (позначка Latest). Для встановлення програми вам потрібен файл із назвою AgencyContractor-win-Setup.exe.
