# Agency Contractor 0.1.59 — Актуальніші дані Telegram-бота і покращені звіти

## Telegram-бот: зарплата і актуальні дані
- **Актуальніші дані по зарплаті** — бот тепер краще підтягує свіжі нарахування з поточних записів, а не лише з історії виплат
- **Показ статусу виплати** — у відповідях по зарплаті тепер видно, чи зарплата вже виплачена, чи ще очікує виплати
- **Дата виплати** — якщо зарплату виплачено, бот показує дату виплати
- **Чіткіше розділення зарплат** — у повній картці працівника окремо показується `Поточна зарплата` і `Остання виплачена зарплата`
- **Краще пояснення даних** — відповіді стали зрозумілішими, коли є нові, але ще не виплачені години або зарплати

## Telegram-бот: нова аналітика по періоду
- **Плинність кадрів за період** — бот уміє показувати прийнятих і звільнених працівників за вибраний період
- **Розбивка по місяцях** — можна отримати підсумок по кількох місяцях, кварталу або іншому діапазону
- **Розбивка по фірмах** — у підсумку доступна статистика по фірмах
- **Net change** — бот рахує чисту зміну кількості працівників за обраний період

## Звіти
- **Нова колонка `Номер візи`** — додано повну підтримку колонки у звіті працівників
- **Колонка доступна у виборі** — `Номер візи` можна ввімкнути через налаштування колонок
- **Колонка реально відображається в таблиці** — виправлено проблему, коли поле було в налаштуваннях, але не з'являлося у самому звіті
- **Підтримка локалізацій** — назву колонки додано для української, англійської, чеської та російської мов

## Автооновлення
- **Підготовлено для каналу оновлень** — реліз `0.1.59` може бути опублікований як наступне автоматичне оновлення для користувачів

## Технічне
- **Без змін у робочих даних** — доробки не додають Telegram-боту можливостей змінювати або видаляти працівників, документи чи зарплати
- **Покращено опрацювання фінансових записів** — логіка вибору актуальної зарплатної інформації стала точнішою
