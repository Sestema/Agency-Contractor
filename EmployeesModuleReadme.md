Налаштування вкладки «Працівники»

Джерело даних
- Дані беруться з файлової системи, не з HTTP API
- Шлях: {RootFolder}\Employers\{FirmName}\Employees\{EmployeeFolder}\employee.json

Вимоги до структури
- RootFolder має бути заданий у налаштуваннях
- Для кожного працівника має бути папка з employee.json
- Фото працівника зберігається як "{FirstName} {LastName} - Photo.jpg" у папці працівника

Очікувана кількість записів
- Відображається кількість папок у Employees, що містять коректний employee.json
- Якщо записів 0, показується порожній стан з кнопкою «Додати працівника»

Перевірка функціоналу
- Пошук працює по ПІБ, номеру паспорту, номеру візи, номеру страховки
- Кнопки «Відкрити», «Редагувати», «Видалити» доступні для кожного запису
- Перегляд працівника відкриває вкладки «Документи» і «Анкета»
