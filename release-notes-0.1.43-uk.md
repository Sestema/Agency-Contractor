# Agency Contractor 0.1.43

Оновлення PDF-шаблонів, даних документів працівника та стабільності встановлення PDF form режиму.

- у PDF-редакторі для native form fields тепер видно, де саме поле знаходиться на сторінці, яке поле вибране, і який текст або тег уже прив'язаний до цього поля;
- покращено заповнення PDF form fields: можна вставляти кілька тегів в один рядок, дописувати звичайний текст праворуч і вставляти теги точно в позицію курсора;
- додано окремі поля `PassportAuthority` і `VisaAuthority` для працівника, майстра додавання працівника, існуючих профілів, історії змін і генерації документів;
- AI-сканування документів тепер зчитує `Orgán vydávající doklad` для паспорта та ID-картки, а також окремий орган видачі для візи;
- додано нові теги шаблонів для органу видачі паспорта та візи;
- якщо для роботи з PDF form fields не встановлена Java, програма тепер сама перевіряє її наявність, пропонує завантажити Eclipse Temurin JDK 21 і запускає інсталятор без `winget`;
- збережено fallback на overlay-режим, щоб редактор PDF залишався працездатним навіть без Java або при проблемах запуску JVM.
