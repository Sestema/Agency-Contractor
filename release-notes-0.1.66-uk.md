# Agency Contractor 0.1.66 — стабільніше збереження зарплат

## Зарплати та фінанси
- **Виправлено збереження годин** — години, введені у фінансах, тепер коректно комітяться з таблиці та записуються в SQLite.
- **Захист від старих дублів** — якщо в `salary_YYYY_MM.db` є старі рядки того самого працівника з іншим шляхом папки, програма бере найновіший запис.
- **Очищення дублів при збереженні** — при записі зарплати старі дублікати того самого `employee_id` у тій самій фірмі прибираються автоматично.
- **Безпечне збереження для двох ПК** — програма продовжує оновлювати рядки працівників через UPSERT, без перезапису всього місяця.

## Панель керування
- **Нові місяці зарплат знову видно** — dashboard тепер бере список місяців із SQLite-файлів `SQLite\Vyplaty\salary_YYYY_MM.db`.
- **Старі `salary_*.json` більше не потрібні для відображення місяців** — панель керування працює з актуальним SQLite-сховищем зарплат.

## Діагностика та тести
- **Додано лог `Salary.Save`** — у логах видно місяць, кількість рядків, dirty-рядки та скільки записів пішло на збереження.
- **Додано регресійні тести** — перевірено збереження зарплат без втрати інших працівників і правильну роботу зі старими дублями.
